CSS Tutorial

 CSS signifie "Cascading Style Sheets" (feuilles de style en cascade).
C'est un langage de style utilisé pour décrire l'apparence et la présentation des documents HTML.
Il permet de contrôler des aspects tels que la mise en forme du texte, la disposition des éléments, les couleurs, les images d'arrière-plan, les animations, etc.
Le CSS est un standard web maintenu par le World Wide Web Consortium (W3C).

Rôle du
CSS dans le
développement web
A

-> Le HTML est responsable de la structure et du contenu d'une page web, tandis que le CSS gère la présentation visuelle.

B

-> La séparation du contenu et de la présentation offre plusieurs avantages

C

-> Facilité de maintenance : les mises à jour de style n'affectent pas le contenu HTML.
-> Flexibilité : un même contenu HTML peut avoir différents styles selon le contexte (impression, mobile, etc.).

D

-> Accessibilité : le contenu reste accessible même sans les styles CSS.
-> Optimisation du code : le HTML reste léger et centré sur le contenu.




Fonctionnement du CSS avec le HTML
01

-> Pour appliquer des styles CSS à une page HTML, vous devez lier un fichier CSS externe ou inclure les styles directement dans la section du document HTML.

02

-> Les navigateurs web analysent le code HTML et CSS, puis appliquent les styles CSS aux éléments HTML correspondants.

03

-> Les styles définis dans le CSS remplacent les styles par défaut du navigateur

Syntaxe de base du CSS


Sélecteurs

  • Les sélecteurs permettent de cibler les éléments HTML auxquels les styles CSS doivent être appliqués.
  • Types d'éléments : div, h1, p, etc. (sélectionnent tous les éléments de ce type).
  • Classes : .nomClasse (sélectionne tous les éléments ayant cette classe).
  • ID : #idElement (sélectionne l'élément unique avec cet identifiant).
  • Sélecteurs universels : * (sélectionne tous les éléments).
  • Sélecteurs descendants : div p (sélectionne tous les éléments <p> à l'intérieur de <div>).
  • Sélecteurs enfants : ul > li (sélectionne les éléments <li> directement enfants de <ul>).
  • Vous pouvez combiner les sélecteurs pour cibler des éléments spécifiques.

    • Propriétés et

    valeurs

  • Une déclaration CSS est composée d'une propriété et d'une valeur, séparées par deux-points :.
  • Exemples de propriétés : color, font-size, margin, background-color, etc.
  • Chaque propriété accepte des valeurs spécifiques (couleurs, tailles, unités, mots-clés, etc.).
  • Plusieurs déclarations sont séparées par un point-virgule ;.

    • Règles CSS

  • Une règle CSS est composée d'un sélecteur suivi d'un bloc de déclarations entre accolades 0.
  • Exemple : h1
  • Les déclarations à l'intérieur des accolades définissent les styles à appliquer aux éléments ciblés par le sélecteur.

    • Commentaires

  • Les commentaires en CSS permettent d'ajouter des annotations et des explications dans le code.
  • Un commentaire en CSS commence par /* et se termine par */.
  • Exemple : /* Ce commentaire explique la règle CSS suivante */
  • Les commentaires sont ignorés par les navigateurs et n'affectent pas le rendu des styles.

    Styles de texte

    Polices

  • font-family : Définit la famille de polices à utiliser (serif, sans-serif, monospace, etc.). Vous pouvez spécifier plusieurs polices en les séparant par des virgules, ce qui permet au navigateur d'utiliser la première police disponible. Exemple : font-family: Arial, sans-serif;


  • font-style : Définit le style de police (normale, italique, oblique). Exemple : font-style: italic;


  • font-weight : Définit la graisse de la police (normal, bold, lighter, bolder, ou valeurs numériques). Exemple : font-weight: bold;


  • font-size : Définit la taille de la police (en pixels, em, rem, pourcentage, etc.). Exemple : font-size: 16px;

    • Couleurs

  • color : Définit la couleur du texte.


  • Couleurs nommées : red, blue, green, etc.


  • Codes hexadécimaux : #FF0000 (rouge), #00FF00 (vert), #0000FF (bleu).


  • Valeurs RGB : rgb(255, 0, 0) (rouge), rgb(0, 255, 0) (vert), rgb(0, 0, 255) (bleu).


  • Valeurs RGBa : Similaire à RGB, mais avec un quatrième paramètre pour l'opacité (rgba(0, 0, 0, 0.5)).

    • Alignement et espacement

  • text-align : Aligne le texte horizontalement (gauche, droite, centre, justifié).


  • text-indent : Indente le premier ligne d'un paragraphe.


  • letter-spacing : Définit l'espacement entre les caractères.


  • word-spacing : Définit l'espacement entre les mots.


  • line-height : Définit la hauteur de ligne du texte.

    • Décoration

  • text-decoration : Ajoute une décoration au texte (souligné, barré, soulignement en pointillés, etc.).


  • text-transform : Transforme la casse du texte (uppercase, lowercase, capitalize).


  • text-shadow : Ajoute une ombre au texte.

    • Styles de boîtes

    Modèle de boîte

  • Chaque élément HTML est considéré comme une boîte rectangulaire.


  • Le modèle de boîte CSS décrit la disposition des différentes parties de cette boîte : le contenu (content), le padding (espace interne), la bordure (border) et la marge (margin).


  • Le calcul des dimensions d'une boîte dépend de la propriété box-sizing (content-box ou border-box).

    • Marges et padding

  • margin : Définit la marge extérieure de la boîte, c'est-à-dire l'espace entre la bordure et les éléments environnants.


  • padding : Définit la marge intérieure de la boîte, c'est-à-dire l'espace entre le contenu et la bordure.


  • Ces propriétés peuvent être définies individuellement (margin-top, padding-left, etc.) ou regroupées (margin: 10px 20px;).

    • Bordures

  • border-style : Définit le style de la bordure (solid, dotted, dashed, etc.).


  • border-width : Définit l'épaisseur de la bordure.


  • border-color : Définit la couleur de la bordure.


  • Vous pouvez définir les styles de bordure individuellement (border-top-style, border-right-width, etc.) ou les regrouper (border: 1px solid black;).

    • Dimensions

  • width : Définit la largeur de la boîte.


  • height : Définit la hauteur de la boîte.


  • max-width et max-height : Définissent les dimensions maximales de la boîte.


  • min-width et min-height : Définissent les dimensions minimales de la boîte.

    Couleurs et arrière-plans


    Couleurs de fond

  • background-color : Définit la couleur d'arrière-plan d'un élément.
  • Vous pouvez utiliser des couleurs nommées (red, blue, etc.), des codes hexadécimaux (#FF0000), des valeurs RGB (rgb(255, 0, 0)) ou RGBa (rgba(255, 0, 0, 0.5)).

    • Dégradés

  • linear-gradient() : Crée un dégradé linéaire.
  • radial-gradient() : Crée un dégradé radial.
  • Vous pouvez spécifier les couleurs et leurs positions dans le dégradé.
  • Exemple : background-image: linear-gradient(to right, red, yellow);

    • Images d'arrière-plan

  • background-image : Définit une image comme arrière-plan d'un élément.
  • Vous pouvez spécifier le chemin vers l'image (url('assets/images/image.jpg.backgroundrepeat_contrle_la_rptition_de_limage_repeat_repeatx_repeaty_norepeat.backgroundposition_dfinit_la_position_de_limage_darrireplan_top_bottom_left_right_center_ou_valeurs_en_pixels_ou_en_pourcentages.backgroundsize_dfinit_la_taille_de_limage_darrireplan_auto_cover_contain_ou_valeurs_en_pixels_ou_en_pourcentages.exemple_backgroundimage_urlfond.jpg'); background-repeat: no-repeat; background-size: cover;

    • Opacité

  • opacity : Définit la transparence d'un élément.

    • La valeur varie de 0 (complètement transparent) à 1 (complètement opaque).

  • Exemple : opacity: 0.5;
