HTML Tutorial

HTML signifie "HyperText Markup Language", ce qui se traduit par "langage de balisage pour l'hypertexte". C'est un langage de balisage standard utilisé pour créer et structurer le contenu des pages web. Le HTML utilise des balises (tags) pour définir les différents éléments d'une page, tels que les titres, les paragraphes, les liens, les images, etc.

La navigation en HTML 

Liens et navigation

Les liens hypertextes sont l'un des éléments clés du World Wide Web. Ils permettent de naviguer d'une page à l'autre, que ce soit sur le même site web ou vers des sites externes. HTML offre plusieurs options pour créer et gérer les liens.

Liens internes (sur la même page)

Les liens internes permettent de naviguer vers des sections spécifiques de la même page web. Ils sont souvent utilisés dans les pages longues comportant plusieurs sections distinctes.

Exemple :

<a href="#section2" > Aller à la section 2 </a>
< !-- ... -- >
<h2> id="section2"> Section 2 </h2>

-> Dans cet exemple, le lien pointe vers l'élément h2 avec l'ID section2 grâce au symbole #.

Liens externes (vers d'autres pages/sites)

Les liens externes permettent de naviguer vers des pages web externes situées sur d'autres sites ou domaines.

Exemple :
<a href="https://www.exemple.com"> Lien vers un site externe </a>

-> L'attribut href spécifie l'URL complète de la page de destination.

Liens d'ancrage (navigation interne)

Les liens d'ancrage permettent de naviguer vers des sections spécifiques d'autres pages du même site web.

Exemple :
< !-- page1.html -- >
<a href="page2.html#section3"> Aller à la section 3 de la page 2 </a>
<!-- page2.html -- >
<h2 id="section3"> Section 3 </h2>

-> Dans cet exemple, le lien pointe vers la page2.html et navigue directement vers l'élément avec l'ID section3.

Ouverture de liens dans un nouvel onglet

Il est possible d'ouvrir une page liée dans un nouvel onglet ou une nouvelle fenêtre du navigateur en utilisant l'attribut target.

Exemple :
<a href="https://www.exemple.com" target="_blank"> Lien s'ouvrant dans un nouvel onglet </a>

-> La valeur _blank de l'attribut target indique au navigateur d'ouvrir le lien dans un nouvel onglet ou une nouvelle fenêtre.

Voici quelques autres attributs utiles pour les liens :

-> title : Affiche une info-bulle lorsque l'utilisateur passe la souris sur le lien.
-> download : Indique au navigateur de télécharger la ressource liée au lieu de la naviguer.
-> rel : Spécifie la relation entre la page actuelle et la page liée (par exemple, rel="noopener noreferrer" pour des raisons de sécurité).

-> Les liens sont essentiels pour la navigation et l'exploration du web. Ils permettent aux utilisateurs de se déplacer d'une page à l'autre et d'accéder à des contenus liés. Leur utilisation correcte et stratégique est cruciale pour une expérience utilisateur réussie.
