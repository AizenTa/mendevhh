HTML Tutorial

HTML signifie "HyperText Markup Language", ce qui se traduit par "langage de balisage pour l'hypertexte". C'est un langage de balisage standard utilisé pour créer et structurer le contenu des pages web. Le HTML utilise des balises (tags) pour définir les différents éléments d'une page, tels que les titres, les paragraphes, les liens, les images, etc.

 L'insertion du contenu multimédia en HTML

  1. I- Images (img) : 
  2. La balise img est utilisée pour insérer une image dans une page web. Voici comment l'utiliser :
                    Exemple : 
                    <img src="chemin/vers/image.jpg" alt="Description de l'image">
        
                    -> L'attribut src spécifie le chemin ou l'URL de l'image à afficher.
                    -> L'attribut alt fournit un texte alternatif pour décrire l'image, important pour l'accessibilité et lorsque l'image ne peut pas être affichée.

Vous pouvez également ajuster la taille de l'image en utilisant les attributs width et height. Cependant, il est préférable de dimensionner les images avec du CSS pour meilleures performances.

                    Exemple : 
                    <img src="image.jpg" alt="Photo de paysage" width="500" height="300">




  1. II- audio :  
                   Exemple :
                    <audio controls>
                        <source src="son.mp3" type="audio/mpeg">
                        Votre navigateur ne prend pas en charge l'élément audio.
                    </audio>

                    -> L'attribut controls affiche les contrôles de lecture par défaut (lecture, pause, volume, etc.).
                    -> L'élément source spécifie le chemin du fichier audio et son type MIME.
                    -> Le texte entre les balises audio et /audio est un message de secours pour les navigateurs qui ne prennent pas en charge cette balise.




  1. III- video : 
                   Exemple :
                   <video width="640" height="360" controls>
                          <source src="video.mp4" type="video/mp4">
                          <source src="video.webm" type="video/webm">
                          Votre navigateur ne prend pas en charge l'élément vidéo.
                    </video>

                    -> Les attributs width et height définissent la taille de la vidéo affichée.
                    -> Vous pouvez fournir plusieurs sources avec différents formats vidéo pour une meilleure compatibilité.
                    -> Le texte entre les balises video et /video est un message de secours pour les navigateurs qui ne prennent pas en charge cette balise.




  1. Il existe d'autres attributs et fonctionnalités pour ces balises, comme autoplay, loop, muted, poster (pour une image de prévisualisation), etc.
  2. Il est important de noter que les formats multimédia pris en charge peuvent varier selon les navigateurs. Il est donc recommandé de fournir des alternatives, comme le texte de secours, pour garantir une expérience utilisateur optimale.
  3. En utilisant ces balises, vous pouvez facilement intégrer des images, du son et des vidéos dans vos pages web, offrant ainsi un contenu riche et engageant pour vos utilisateurs.Delivery.
