HTML Tutorial

HTML signifie "HyperText Markup Language", ce qui se traduit par "langage de balisage pour l'hypertexte". C'est un langage de balisage standard utilisé pour créer et structurer le contenu des pages web. Le HTML utilise des balises (tags) pour définir les différents éléments d'une page, tels que les titres, les paragraphes, les liens, les images, etc.

Commencer

Rôle du HTML dans le développement web :

Le HTML est la pierre angulaire du développement web. C'est le langage qui fournit la structure et le contenu de base d'une page web. Sans HTML, il serait impossible d'afficher du texte, des images ou tout autre contenu sur un site web. Le HTML est donc essentiel et joue un rôle fondamental dans la création de pages web.


Anatomie d'une page HTML (structure de base) :

- Doctype : C'est une déclaration au début du document qui indique la version du HTML utilisée.
- Élément : C'est l'élément racine qui englobe tout le contenu de la page web. 
- Élément : Cette section contient les métadonnées de la page, telles que le titre, les liens vers les feuilles de style CSS, les scripts JavaScript, etc. Ces informations ne sont pas visibles dans le navigateur mais sont essentielles pour le bon fonctionnement de la page. 

Éléments HTML de base

Balises d'en-tête (<head>) et de corps (<body>)

<head> 
 Cette balise contient les informations métadonnées de la page qui ne sont pas affichées dans le navigateur, comme le titre de la page, les liens vers les feuilles de style CSS, les scripts JavaScript, les métadonnées pour les moteurs de recherche, etc.

<body> 
Cette balise englobe tout le contenu visible de la page web, comme le texte, les images, les liens, les formulaires, etc.

Exemples
Titres (<h1> à <h6>)

Les titres sont des éléments importants pour structurer le contenu d'une page. Il existe six niveaux de titres allant de <h1> (le plus important) à <h6> (le moins important). Ces balises sont généralement utilisées pour définir les en-têtes et sous-en-têtes d'une page.

Exemple:
<h1> Titre principal </h1>
<h2> Sous-titre </h2>

Paragraphes (<p>)

La balise <p> est utilisée pour définir un paragraphe de texte. Chaque paragraphe doit être encapsulé dans une balise <p>.

Exemple:
<p> Ceci est un paragraphe de texte. </p>
<p> Ceci est un autre paragraphe. </p>

Liens (<a>)

La balise <a> (ancre) permet de créer des liens hypertextes. L'attribut href spécifie l'URL de destination du lien.

Exemple:
<a href="https://www.example.com">Lien vers un site externe</a>

<a href="about.html"> Lien vers une page du même site </a>

Listes ordonnées (<ol>) et non ordonnées (<ul>)

<ol> (Ordered List) permet de créer une liste ordonnée, avec des éléments numérotés.
<ul> (Unordered List) permet de créer une liste non ordonnée, avec des puces.
Les éléments de la liste sont définis avec la balise <li>.

Exemples:
<ol> 
      <li> Premier élément </li>
      <li> Deuxième élément </li>
</ol>

    Attributs de base (src, href, alt, etc.)

    src : Utilisé avec la balise <img> pour spécifier le chemin ou l'URL de l'image à afficher.

    href : Utilisé avec la balise <a> pour spécifier l'URL de destination d'un lien.

    alt : Fournit un texte alternatif pour les images, important pour l'accessibilité.

    title : Affiche une info-bulle lorsque l'utilisateur passe la souris sur un élément.


    target : Utilisé avec les liens <a> pour spécifier où ouvrir la page liée (même fenêtre, nouvelle fenêtre/onglet).

      Images (<img>)

      La balise <img> permet d'insérer une image dans une page web. L'attribut src spécifie le chemin ou l'URL de l'image, et l'attribut alt fournit un texte alternatif pour les navigateurs qui ne peuvent pas afficher l'image.

      Exemples:
      < img src="logo.png" alt="Logo de l'entreprise">

      Formatage de texte

      Description :

      Le HTML offre plusieurs balises pour mettre en forme et styliser le texte d'une page web. Ces balises permettent de mettre en évidence certaines parties du texte, de le rendre plus lisible et d'ajouter des effets visuels.

      Balises pour le gras (<b>, <strong>)

      <b> 
      Cette balise permet de mettre en gras un texte. Elle est principalement utilisée à des fins de mise en forme visuelle.

                      Exemple : <p> Ceci est un <b>texte en gras</b>.</p>

      <strong>
      Cette balise est similaire à <b> , mais elle est sémantiquement utilisée pour indiquer que le texte mis en gras a une importance particulière ou une forte intensité.
              

                      Exemple : <p> Attention, <strong> ce texte est important</strong>.</p>




      Balises pour l'italique (<i>, <em>)

      <i>
      Cette balise permet de mettre en italique un texte. Elle est principalement utilisée à des fins de mise en forme visuelle.


                      Exemple : <p> Ceci est un <i>texte en italique</i>.</p>


      <em>

      Cette balise est similaire à <i>, mais elle est sémantiquement utilisée pour indiquer que le texte en italique a une emphase particulière ou une importance.

                      Exemple : <p>Je<em>dois</em> finir ce travail aujourd'hui.</p>

      Autres balises de formatage

      <sub> : Permet d'écrire un texte en indice (subscript).
      Exemple :       <p>Formule chimique : H<sub>2</sub>0</p>

      <sup> :  Permet d'écrire un texte en exposant (superscript).
      Exemple : <p>Équation : x<sup>2</sup> + y<sup>2</sup> = z <sup>2</sup></p>

      <br> :  Insère un saut de ligne (line break) dans le texte.
      Exemple : <p> Ligne 1 <br> Ligne 2 </p>

      <hr> : Insère une ligne horizontale (horizontal rule) pour séparer visuellement des sections de contenu.
      Exemple : 
      <h2> Section 1 </h2>
      <p>Contenu de la section 1.</p>
      <hr>
      <h2>Section 2</h2>
      <p>Contenu de la section 2.</p>

      Ces balises de formatage permettent d'ajouter des styles visuels au texte et d'améliorer la lisibilité et la présentation du contenu. Cependant, il est important de les utiliser de manière appropriée et de privilégier les balises sémantiques (<strong> et <em>)  lorsque cela est possible pour une meilleure accessibilité et un meilleur référencement.

      Structure de base d'une page HTML

      Toute page HTML suit une structure standard composée d'éléments essentiels. Cette structure fournit un cadre pour organiser le contenu et les métadonnées de la page. Les trois principaux éléments qui constituent la base d'une page HTML sont.

       Élément racine html

      - L'élément html est l'élément racine qui englobe tout le contenu d'une page web.

      - Il définit le début et la fin du document HTML.

      - Tous les autres éléments HTML doivent être placés à l'intérieur des balises html et /html.

      Élément head

      - L'élément head contient les métadonnées d'une page web, qui ne sont pas directement visibles dans le navigateur.

      - Ces métadonnées fournissent des informations importantes pour les navigateurs, les moteurs de recherche et d'autres services.

      Les éléments couramment placés dans head incluent:
      - <title> : Définit le titre de la page, affiché dans l'onglet du navigateur.

      - <meta> : Fournit des métadonnées supplémentaires, comme la description de la page, les mots-clés, l'encodage des caractères, etc.

      - <link> : Permet de lier des ressources externes, comme les feuilles de style CSS.

      - <script> : Permet d'inclure des scripts JavaScript pour ajouter de l'interactivité à la page.

      Élément body

      - L'élément body contient tout le contenu visible de la page web, comme le texte, les images, les liens, les formulaires, etc.

      - C'est ici que le contenu principal de la page est placé, en utilisant les différentes balises HTML que nous avons vues précédemment (titres, paragraphes, listes, images, etc.).

      - Le contenu à l'intérieur de body est affiché dans la fenêtre du navigateur lorsque la page est chargée.

      Voici un exemple de structure de base d'une page HTML avec ces trois éléments clés :

      <html>

       <!DOCTYPE html>

        <html>

         <head>

          <titleTitre de la page/title>

           <meta charset="UTF-8">

           <link rel="stylesheet" href="styles.css">

          </head>

          <body>

         <h1>Bienvenue sur mon site</h1>

        <p>Ceci est un paragraphe de contenu.</p>

       <img src="image.jpg" alt="Description de l'image">

       </body>

      </html>

      Sémantique et structure du document

      La sémantique fait référence à l'utilisation d'éléments HTML appropriés pour décrire la structure et le contenu d'une page web de manière significative. Elle permet d'améliorer l'accessibilité, le référencement et la maintenabilité du code.

      Importance de la sémantique

      Accessibilité : Les technologies d'assistance telles que les lecteurs d'écran peuvent mieux comprendre et interpréter le contenu grâce à des éléments sémantiques appropriés.

      Référencement : Les moteurs de recherche accordent une importance particulière à la structure sémantique du contenu pour mieux comprendre et classer les pages web.

      Maintenabilité : Un code sémantique est plus facile à lire, à comprendre et à mettre à jour pour les développeurs.

      Futureproof : Les éléments sémantiques sont conçus pour une meilleure compatibilité avec les futures versions et les nouvelles fonctionnalités du web.

      Éléments sémantiques de base

      HTML5 a introduit de nouveaux éléments sémantiques pour mieux structurer le contenu d'une page web. Voici quelques-uns des plus couramment utilisés:

      header : Définit l'en-tête d'une section ou de la page entière (logo, navigation, etc.).
      nav : Englobe les liens de navigation principaux du site.
      main : Représente le contenu principal unique d'une page.
      section : Définit une section thématique du contenu.
      article : Encapsule un contenu indépendant et autonome (comme un article de blog ou une actualité).
      aside : Représente un contenu secondaire lié au contenu principal (publicités, notes de bas de page, etc.).
      footer : Définit le pied de page d'une section ou de la page entière (informations de copyright, liens, etc.).

      Voici un exemple de structure sémantique pour une page web simple :

      <html>
      <header>
          <nav>
              <!-- Liens de navigation -- >
          </nav>
      </header>

      <main>
          <section>
              <h1>Section principale</h1>
              <article>
                  <h2>Article 1</h2>
                  <!-- Contenu de l'article -- >
          </article>
          <article>
              <h2>Article 2</h2>
              <!-- Contenu de l'article -- >    
              </article>
          </section>
          <aside>
              <!-- Contenu secondaire -- >
          </aside>
      </main>

      <footer>
          <!-- Pied de page -- >
      </footer>

       Les listes en HTML 

      Listes

      Les listes sont un moyen pratique d'organiser et de présenter des éléments liés dans un format structuré.
      HTML offre deux types principaux de listes :
      les listes non ordonnées et les listes ordonnées.
      Il est également possible d'imbriquer des listes les unes dans les autres pour créer des structures hiérarchiques.

      Listes non ordonnées <ul>

      Les listes non ordonnées (ul pour "unordered list") sont utilisées pour représenter un groupe d'éléments sans ordre particulier. Chaque élément est précédé d'une puce par défaut.

      -> Exemple : 

      <ul>          
          <li>Pomme</li>
          <li>Banane</li>
          <li>Orange</li>
      </ul>

      Listes ordonnées <ol>

      Les listes ordonnées (<ol> pour "ordered list") sont utilisées pour représenter un groupe d'éléments dans un ordre spécifique, généralement numéroté.

      -> Exemple : 

      <ol>
      <li>Étape 1</li>
      <li>Étape 2</li>
      <li>Étape 3</li>
      </ol>

      Listes imbriquées

      Il est possible d'imbriquer des listes les unes dans les autres pour créer des structures hiérarchiques. Une liste imbriquée est une liste placée à l'intérieur d'un élément li d'une autre liste.

      ->Exemple : 

      <ul>
            <li>Fruits
      <ul>
           <liPomme</li>
             <li>Banane</li>
      </ul>
          </li>
          <li>Légumes
           <ol>
              <li>Carotte</li>
               <li>Brocoli</li>
             </ol>
           </li>
      </ul>

      Description

      -> Dans cet exemple, nous avons une liste non ordonnée principale avec deux éléments. Le premier élément contient une liste non ordonnée imbriquée de fruits, et le deuxième élément contient une liste ordonnée imbriquée de légumes.

      -> Vous pouvez contrôler le type de puce ou de numérotation utilisé pour chaque liste en utilisant les attributs type ou start. Par exemple, ol type="A" créera une liste ordonnée avec des lettres majuscules.

      -> Les listes sont très utiles pour présenter des informations de manière structurée et hiérarchique, que ce soit pour des instructions, des menus, des descriptions ou tout autre contenu organisé.

      La navigation en HTML 

      Liens et navigation

      Les liens hypertextes sont l'un des éléments clés du World Wide Web. Ils permettent de naviguer d'une page à l'autre, que ce soit sur le même site web ou vers des sites externes. HTML offre plusieurs options pour créer et gérer les liens.

      Liens internes (sur la même page)

      Les liens internes permettent de naviguer vers des sections spécifiques de la même page web. Ils sont souvent utilisés dans les pages longues comportant plusieurs sections distinctes.

      Exemple :

      <a href="#section2" > Aller à la section 2 </a>
      < !-- ... -- >
      <h2> id="section2"> Section 2 </h2>

      -> Dans cet exemple, le lien pointe vers l'élément h2 avec l'ID section2 grâce au symbole #.

      Liens externes (vers d'autres pages/sites)

      Les liens externes permettent de naviguer vers des pages web externes situées sur d'autres sites ou domaines.

      Exemple :
      <a href="https://www.exemple.com"> Lien vers un site externe </a>

      -> L'attribut href spécifie l'URL complète de la page de destination.

      Liens d'ancrage (navigation interne)

      Les liens d'ancrage permettent de naviguer vers des sections spécifiques d'autres pages du même site web.

      Exemple :
      < !-- page1.html -- >
      <a href="page2.html#section3"> Aller à la section 3 de la page 2 </a>
      <!-- page2.html -- >
      <h2 id="section3"> Section 3 </h2>

      -> Dans cet exemple, le lien pointe vers la page2.html et navigue directement vers l'élément avec l'ID section3.

      Ouverture de liens dans un nouvel onglet

      Il est possible d'ouvrir une page liée dans un nouvel onglet ou une nouvelle fenêtre du navigateur en utilisant l'attribut target.

      Exemple :
      <a href="https://www.exemple.com" target="_blank"> Lien s'ouvrant dans un nouvel onglet </a>

      -> La valeur _blank de l'attribut target indique au navigateur d'ouvrir le lien dans un nouvel onglet ou une nouvelle fenêtre.

      Voici quelques autres attributs utiles pour les liens :

      -> title : Affiche une info-bulle lorsque l'utilisateur passe la souris sur le lien.
      -> download : Indique au navigateur de télécharger la ressource liée au lieu de la naviguer.
      -> rel : Spécifie la relation entre la page actuelle et la page liée (par exemple, rel="noopener noreferrer" pour des raisons de sécurité).

      -> Les liens sont essentiels pour la navigation et l'exploration du web. Ils permettent aux utilisateurs de se déplacer d'une page à l'autre et d'accéder à des contenus liés. Leur utilisation correcte et stratégique est cruciale pour une expérience utilisateur réussie.

       L'insertion du contenu multimédia en HTML

      1. I- Images (img) : 
      2. La balise img est utilisée pour insérer une image dans une page web. Voici comment l'utiliser :
                          Exemple : 
                          <img src="chemin/vers/image.jpg" alt="Description de l'image">
              
                          -> L'attribut src spécifie le chemin ou l'URL de l'image à afficher.
                          -> L'attribut alt fournit un texte alternatif pour décrire l'image, important pour l'accessibilité et lorsque l'image ne peut pas être affichée.

      Vous pouvez également ajuster la taille de l'image en utilisant les attributs width et height. Cependant, il est préférable de dimensionner les images avec du CSS pour meilleures performances.

                          Exemple : 
                          <img src="image.jpg" alt="Photo de paysage" width="500" height="300">




      1. II- audio :  
                         Exemple :
                          <audio controls>
                              <source src="son.mp3" type="audio/mpeg">
                              Votre navigateur ne prend pas en charge l'élément audio.
                          </audio>

                          -> L'attribut controls affiche les contrôles de lecture par défaut (lecture, pause, volume, etc.).
                          -> L'élément source spécifie le chemin du fichier audio et son type MIME.
                          -> Le texte entre les balises audio et /audio est un message de secours pour les navigateurs qui ne prennent pas en charge cette balise.




      1. III- video : 
                         Exemple :
                         <video width="640" height="360" controls>
                                <source src="video.mp4" type="video/mp4">
                                <source src="video.webm" type="video/webm">
                                Votre navigateur ne prend pas en charge l'élément vidéo.
                          </video>

                          -> Les attributs width et height définissent la taille de la vidéo affichée.
                          -> Vous pouvez fournir plusieurs sources avec différents formats vidéo pour une meilleure compatibilité.
                          -> Le texte entre les balises video et /video est un message de secours pour les navigateurs qui ne prennent pas en charge cette balise.




      1. Il existe d'autres attributs et fonctionnalités pour ces balises, comme autoplay, loop, muted, poster (pour une image de prévisualisation), etc.
      2. Il est important de noter que les formats multimédia pris en charge peuvent varier selon les navigateurs. Il est donc recommandé de fournir des alternatives, comme le texte de secours, pour garantir une expérience utilisateur optimale.
      3. En utilisant ces balises, vous pouvez facilement intégrer des images, du son et des vidéos dans vos pages web, offrant ainsi un contenu riche et engageant pour vos utilisateurs.Delivery.
